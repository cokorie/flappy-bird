# Flappy Bird

### __Flappy Bird__
Enjoy a desktop rebuild of the cult classic game, Flappy Bird. This version is similar to the original, by moving the Flappy Bird past the pipes by jumping through them.

Use the Space bar on your computer to manuever through the game, and see how high your score can go!

### __Getting Started__
Please click into the following website link to check out my rebuild of Flappy Bird:  
[Flappy Bird](https://cokorie.github.io/flappy-bird/)

### __Technologies Used__
##### _HTML_
  The HTML code is used to house the: 
  * Game title
  * Game board

###### _CSS_ 
  The CSS code is used to house the styling of the:
  * Game window
  * Game board
  * Game background

###### _JavaScript_ 
  The JaveScript code is used to house the functions & logic of:
  * Game board
  * Game sounds
  * Player positions
  * Pipe positions
  * Ground positioning
  * Background positioning
  * Game speed
  * Player jumping
  * Game reset after finish
  * Player collisions with pipes, sky and ground

###### _GitHub_ 
  GitHub is used to remotely house:
  * Game files including the HTML, CSS, JavaScript and media files used to build this version of the game
