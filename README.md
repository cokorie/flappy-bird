# Flappy Bird

### __Flappy Bird__
Enjoy a desktop rebuild of the cult classic game, Flappy Bird. This version is similar to the original, by moving the Flappy Bird past the pipes by jumping through them.

Use the Space bar on your computer to manuever through the game, and see how high your score can go!

### __Technologies Used__
* HTML
* CSS
* JavaScript
* GitHub
